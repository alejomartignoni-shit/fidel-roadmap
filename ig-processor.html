How it works

Full Pipeline

From a ClickUp task with an Instagram URL to 10 production-ready slide briefs — fully automated.

1

Alejo creates a ClickUp task

Pastes the Instagram URL in the task name or description. Sets status to request.

ClickUp
2

ig-processor picks it up within 2 min

Polls 7 ClickUp lists every 2 minutes. Locks task to in progress to prevent double-processing.

PM2 · id 12
3

Download carousel images via Playwright

Best-effort scrape. If Instagram blocks it, falls back to caption text pasted in the task.

Playwright
4

Claude CLI analyzes each slide

Reads each image with the Read tool. Captures verbatim text, element positions, layout structure, emotional hook, and visual mechanic. 3s delay between calls to avoid throttling.

Claude CLI · --allowedTools Read
5

Raw vision posted as ClickUp comment

The exact analysis of what the AI saw in each slide — permanent reference. Never mixed into the description.

ClickUp comment
6

Generate 10-section brand reframe

Claude CLI takes all slide visions and generates: OBJETIVO / NARRATIVA / HOOKS / SECUENCIA / COPY / CTA / TONALIDAD / PRODUCCION / CAPTION / HASHTAGS.

Claude CLI · 300s timeout
7

Write reframe to task description

Only the brand-adapted content goes here. Renamed to first 5 words of COPY SUGERIDO.

ClickUp description
8

Create one subtask per slide

Each slide gets a unique production brief generated from its own vision output. TOFU / MOFU / BOFU copy, fully written. 3s delay between each.

Claude CLI ClickUp subtask
9

Status → In Review

Task is handed off to the team. Paulo reviews, Facu edits, Alejo approves.

Done

Content Strategy

Content Funnel

Every piece of content — every slide brief — maps to one of these three stages.

Transformation Not running ads → Running ads
TOFU · Top of Funnel
Anyone in business, sales, or self-employment
Universal human truth — the grind of chasing vs the peace of having a system that works. No niche mention, no industry reference. A plumber, a freelancer, a sales rep — everyone should stop scrolling.
Rule: zero niche. pure insight.
↓
MOFU · Middle of Funnel
People not running ads but feeling the gap
Makes them feel the cost of NOT having a system working for them. "What if" framing. The pain of working hard with no engine behind it. Soft bridge — curiosity, not pitch.
Rule: name the gap, not the product.
↓
BOFU · Bottom of Funnel
Contractors (roofers / plumbers / electricians / HVAC), agents, wholesalers
Specific and direct. Names the exact transformation — what changes the day they start running ads. Shows exactly what they're leaving on the table right now.
Rule: name the niche. name the outcome.
Per-Slide Briefs

Subtask Structure

Every carousel slide generates one ClickUp subtask with this exact format.

SLIDE N — ClickUp Subtask
THE MECHANIC
What this slide does psychologically — contrast, status signal, identity mirror, list reveal, curiosity gap…
THE MEANING
Each element decoded: [element] → communicates [benefit / pain / insight]. Not what it looks like — what it makes the viewer feel.
THE REFRAME
Concrete filmable scene with the ICP as character. Layout matches the original exactly. Viewer sees themselves — not Fidel.
WHY IT WORKS
One sentence: the exact moment of recognition the ICP feels when they see this slide.
TOFU Copy
Universal. No niche. Any person scrolling should feel it in their gut.
MOFU Copy
Gap + cost of not having a system. Makes them feel what they're missing.
BOFU Copy
ICP-specific. Contractor or agent. Direct. Names the exact outcome.
Technical Stack

How It's Built

Two AI layers. Zero API credits. Runs 24/7 on PM2.

🤖
Claude CLI — Vision
claude -p "..." --allowedTools Read --add-dir <dir>

Reads image files natively using the Read tool. 3s delay between calls. 300s timeout for full analysis.
Free · Subscription
💬
Claude CLI — Text
claude -p "system + user prompt"

Generates 10-section brand analysis and per-slide TOFU/MOFU/BOFU briefs. No API key needed.
Free · Subscription
🦙
Ollama — Fallback
Local model (qwen3:8b) as fallback when Claude CLI fails. Runs on localhost. No internet, no cost.
Local Fallback
⚙️
PM2 — Process Manager
PM2 id 12. Polls 7 ClickUp lists every 2 minutes. Auto-restarts on crash. Survives Windows reboots via Task Scheduler.
24/7 · Always On
